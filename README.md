# patient-system-cpp

Patient Management System (C++)

Përshkrimi i Projektit

Ky është një projekt i thjeshtë në C++ për menaxhimin e pacientëve dhe termineve në një sistem klinik. Projekti është zhvilluar për qëllime akademike dhe demonstron përdorimin e strukturave bazë të programimit, klasave dhe menaxhimit të të dhënave.

Funksionalitetet

* Shtimi i pacientëve
* Shfaqja e listës së pacientëve
* Fshirja e pacientëve
* Shtimi i termineve (appointments)
* Shfaqja e termineve
* Menu interaktive në console

Teknologjitë e përdorura

* C++
* STL (vector, string)
* Console Application


Si të ekzekutohet projekti

1. Klono repository

git clone <linku-i-repository>

2.Kompailo kodin

g++ main.cpp -o app

3.Ekzekuto programin

./app
